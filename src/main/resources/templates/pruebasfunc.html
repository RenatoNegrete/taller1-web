Pruebas funcionales

Para hacer las pruebas funcoinales se hacen peticiones a los endpoint del backend a traves de Postman.

Peticiones de CRUD

Entidad Finca

Get Fincas

    • Metodo: GET /api/finca
      Descripcion: Obtener todas las fincas del sistema.
      No necesita un cuerpo.
      Resultado:

    • Get Finca by id

    • Metodo: GET /api/finca/id
      Descripcion: Obtener una finca del sistema por su id.
      No necesita un cuerpo.
      Resultado:
      Posibles errores: Error 404 Not Found.

    • Post Finca

    • Metodo: POST /api/finca
      Descripcion: Crear una finca en el sistema.
      Cuerpo:
      Resultado:
      Posibles errores: Error 404 Not Found.

    • Put Finca

    • Metodo: PUT /api/finca
      Descripcion: Actualiza una finca del sistema.
      Cuerpo:
      Resultado:
      Posibles errores: Error 404 Not Found.

    • Delete Finca

    • Metodo: DELETE /api/finca/id
      Descripcion: Eliminar una finca del sistema por su id.
      No necesita un cuerpo.
      Resultado: 200 OK.
      Posibles errores: Error 404 Not Found.
